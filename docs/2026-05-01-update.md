## 変更詳細

### #1: `claude project purge` コマンド（🟡）

**変更元 CC**: v2.1.126
**原因**: プロジェクトデータの整理に使える新 CLI コマンドが追加された。
`claude project purge [path]` でプロジェクトに紐づくデータを削除できる。
フラグ: `--dry-run`（削除せず確認）、`-y/--yes`（確認スキップ）、`-i/--interactive`（対話式）、`--all`（全プロジェクト）

**Before**: README の Tips に `claude project purge` の記述なし

**After**: README Tips セクションの既存の `claude plugin prune` の行の後に追記:
```
- **Run `claude project purge` to clean up**: Remove stored project data; use `--dry-run` to preview, `--interactive` to select items (v2.1.126+)
```

### #3: `claude_code.skill_activated` OTel イベント（⚪）

スキル実行時に `claude_code.skill_activated` イベント（`invocation_trigger` 属性付き）が
OpenTelemetry で発行されるようになった。OTel 固有機能のため、テンプレート変更不要。

### #4: v2.1.126 バグ修正群（⚪）

主な修正:
- `allowManagedDomainsOnly`/`allowManagedReadPathsOnly` が無視されていたセキュリティ修正
- 2000px 超の画像をペースト時に自動縮小
- Mac スリープ後の "Stream idle timeout" 修正
- Windows: クリップボード書き込みが EDR/SIEM テレメトリに露出する問題の修正
- Cursor/VS Code 1.92–1.104 でのトラックパッドスクロール修正
- Agent SDK での並列バッチ中の不正なツール名によるハング修正
- PowerShell 7 の検出改善（Microsoft Store, MSI, .NET グローバルツール対応）

いずれもテンプレートへの変更は不要。
