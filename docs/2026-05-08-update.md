## 変更詳細

### #1: hooks.md — effort.level フィールドと $CLAUDE_EFFORT 環境変数 (v2.1.133+)

**Before:** `$CLAUDE_EFFORT` はフックドキュメントに記載なし

**After:** Advanced Features セクションに新項目追加:

```markdown
### Hook Input: `effort.level` フィールド と `$CLAUDE_EFFORT` 環境変数 (v2.1.133+)

すべてのフック入力 JSON に `effort.level` フィールドが含まれる。
`command` 型フックおよび Bash サブプロセスでは `$CLAUDE_EFFORT` 環境変数として参照可能。

値: `"low"` / `"medium"` / `"high"` / `"xhigh"`
```

### #2: README.md — worktree.baseRef 設定 (v2.1.133+)

```
| `worktree.baseRef` | ワークツリーのブランチ起点（`"fresh"`: ベースブランチから / `"head"`: 現在の HEAD から）（v2.1.133+） |
```

### #3: README.md — parentSettingsBehavior admin キー (v2.1.133+)

設定テーブルに追加:

| `parentSettingsBehavior` | admin 設定の結合方式（`"first-wins"`: 最上位優先 / `"merge"`: 全階層をマージ）（v2.1.133+） |
