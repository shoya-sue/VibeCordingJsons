## 変更詳細

### #1: ECC v1.9.0 → v1.10.0（前回更新の漏れ）

前回の 2026-04-08 update では ECC v1.10.0 が最新と記載されたが、`CLAUDE_PLUGIN_ROOT` パスの更新アクションが欠落していた。

**変更前:**
```json
"CLAUDE_PLUGIN_ROOT": "${HOME}/.claude/plugins/cache/everything-claude-code/everything-claude-code/1.9.0"
```
**変更後:**
```json
"CLAUDE_PLUGIN_ROOT": "${HOME}/.claude/plugins/cache/everything-claude-code/everything-claude-code/1.10.0"
```

---

### #2: WorktreeCreate の `worktreePath` 返却（v2.1.84）

`type: "http"` フックが `hookSpecificOutput.worktreePath` を返すことでワークツリーパスを指定できる。既存の注記が欠落していた。

**hooks.md の WorktreeCreate 行を更新:**

```
| `WorktreeCreate` | ワークツリー作成時（`type:"http"` で `worktreePath` 返却可、v2.1.84+） | No |
```

---

### #3: `/reload-plugins` と `/effort max`（v2.1.97 / v2.1.98）

- `/reload-plugins` — プラグイン提供スキルを再起動なしで再読み込み（v2.1.97）
- `/effort max` — 最大思考レベルを設定（v2.1.98 で `/effort` コマンドに追加）

**full/CLAUDE.md の Slash Commands セクションを更新:**
```
- `/effort low|medium|high|max` — Set thinking level. `/effort auto` to reset
- `/reload-plugins` — プラグインスキルを再起動なしで再読み込み
```

---

### #4: `refreshInterval` 設定（v2.1.97）

ステータスラインコマンドを N 秒ごとに自動再実行する設定。

**README.md の env 変数テーブルに追加:**
```
| `refreshInterval` | ステータスラインの自動更新間隔（秒） | `30` |
```
