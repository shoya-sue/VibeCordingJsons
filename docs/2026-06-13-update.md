## 変更詳細

### #1 `wheelScrollAccelerationEnabled`（v2.1.174+）

fullscreen モードでマウスホイールのスクロール加速を無効化する settings キー。テンプレは既に `/scroll-speed`（v2.1.139+）と `autoScrollEnabled` を文書化しているため、設定表とスラッシュコマンド注記に併記。

- README「Other Settings」表に行追加
- `template/CLAUDE.md` の `/scroll-speed` 行末に「settings.json の `wheelScrollAccelerationEnabled: false` でホイール加速を無効化（v2.1.174+）」を併記

### #2 `enforceAvailableModels`（v2.1.175+、managed）

有効時、`availableModels` allowlist が **Default モデル**も制約する（disallow に解決される Default は最初の allowed モデルにフォールバック）。user/project 設定で managed の `availableModels` を広げられなくなる。README 設定リファレンスの managed 設定群に追記。

### #3 `footerLinksRegexes`（v2.1.176+）

フッター行に regex マッチの link badge を出す設定。user/managed どちらでも設定可。README 設定リファレンスに追記。

### #4 session title 言語ピン（v2.1.176+）

session title が会話の言語で自動生成されるようになり、`language` 設定で特定言語にピン留めできる。テンプレは `language: japanese` を設定済みのため、session title も日本語で生成される。

- README「Other Settings」`language` 行に「session title もこの言語で生成（v2.1.176+）」を追記
- `template/CLAUDE.md` の voice/language 注記に同旨を追記

### #5 hook `if` 条件の Read/Edit/Write パスパターン（v2.1.176+ 修正）

`Edit(src/**)`, `Read(~/.ssh/**)`, `Read(.env)` のような documented パターンが hook の `if` 条件で正しくマッチするよう修正。`hooks.md` の Conditional Filtering（`if` field）節は現状 `Bash(git *)` 例のみなので、Read/Edit/Write のパスパターンも使える旨を追記。
