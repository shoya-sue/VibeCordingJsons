## 変更詳細

### #1 template/.mcp.json
```json
"plaud": {
  "command": "npx",
  "args": ["-y", "@plaud-ai/mcp@latest"],
  "env": {
    "PATH": "NODEJS_BIN_DIR:/opt/homebrew/bin:/usr/local/bin:/usr/bin:/bin",
    "DO_NOT_TRACK": "1"
  }
}
```
- API キーなし（OAuth）。初回のみ `npx -y @plaud-ai/mcp@latest install` でログイン → `~/.plaud/` にトークン保存。
- `DO_NOT_TRACK=1` で PostHog テレメトリ無効化。

### #2 template/.codex/config.toml
```toml
[mcp_servers.plaud]
command = "npx"
args = ["-y", "@plaud-ai/mcp@latest"]
startup_timeout_sec = 30
tool_timeout_sec = 60

[mcp_servers.plaud.env]
PATH = "NODEJS_BIN_DIR:/opt/homebrew/bin:/usr/local/bin:/usr/bin:/bin"
DO_NOT_TRACK = "1"
```

### #3 settings.json permissions.allow
`"mcp__excalidraw__*"` 群に並べて `"mcp__plaud__*"` を追加。

### #5 AGENTS.md

→ `- 7 MCP servers: obsidian, context7, playwright, deepwiki, excalidraw, github, plaud (optional — Plaud device owners; OAuth, unauth = tools hidden)`
