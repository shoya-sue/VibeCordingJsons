## 変更詳細

### #1: genshijin グローバルインストール（🟡 推奨）

**カテゴリ**: プラグイン

Julius Brussee 作の caveman（英語版）を日本語向けに最適化したトークン圧縮スキル。
ベンチマーク（30タスク）: 通常 1,483 → genshijin 296 トークン（80%削減）。
敬語体言止め化・クッション言葉削除・冗長助詞圧縮の3レイヤーで動作。

```
claude plugin marketplace add InterfaceX-co-jp/genshijin
claude plugin install genshijin
```

---

### #2: `/genshijin` コマンド追加（🟡 推奨）

**カテゴリ**: CLAUDE.md / Slash Command

**Before:**
```
- `/effort low|medium|high|xhigh|max` — Set thinking level. `/effort auto` to reset
- `/memory` — Manage auto-memory
```

**After:**
```
- `/effort low|medium|high|xhigh|max` — Set thinking level. `/effort auto` to reset
- `/genshijin [丁寧|極限]` — 応答トークン80%削減（原始人モード）。`丁寧`でビジネス向け、`極限`で最大圧縮。`原始人やめて`で解除
- `/memory` — Manage auto-memory
```
