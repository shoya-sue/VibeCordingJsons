## 変更詳細

### #1: `/effort xhigh` レベル追加（🟡 推奨）

**対象バージョン**: CC v2.1.111
**カテゴリ**: CLAUDE.md / Slash Command ドキュメント

v2.1.111 で Claude Opus 4.7 向けに `xhigh` 努力レベルが追加。
`high` と `max` の中間に位置する新しいレベル。
CLAUDE.md のコマンド記述を更新してユーザーに認知させる（documentation only）。

**Before:**
```
- `/effort low|medium|high|max` — Set thinking level. `/effort auto` to reset
```

**After:**
```
- `/effort low|medium|high|xhigh|max` — Set thinking level. `/effort auto` to reset
```

### #4: Claude Opus 4.7（⚪ スキップ）

**対象バージョン**: CC v2.1.111
**カテゴリ**: モデル / performance.md

v2.1.111 で Claude Opus 4.7 が Auto モードで利用可能（Max サブスクライバー向け）。

変更予定:

### #5: `effortLevel: "xhigh"` in settings.json（⚪ スキップ）

**対象バージョン**: CC v2.1.111
**カテゴリ**: settings.json キー / performance.md

performance.md の設定例 `effortLevel: "low" | "medium" | "high"` に `xhigh` を追加すべきだが、
CC 2.1.92 で `effortLevel: "xhigh"` を設定するとエラーになる可能性があるため version gate。

変更予定（version gate 解除後）:
```
effortLevel: "low" | "medium" | "high" | "xhigh"
```
