# VibeCording Update — 2026-05-18

## 変更点サマリー

Claude Code v2.1.143 (2026-05-15) の変更を反映。Stop フックの 8 連続ブロック上限 (`CLAUDE_CODE_STOP_HOOK_BLOCK_CAP`)、Bedrock/Vertex/Foundry 利用時の Windows PowerShell ツール有効化のオプトアウト (`CLAUDE_CODE_USE_POWERSHELL_TOOL` / `CLAUDE_CODE_POWERSHELL_RESPECT_EXECUTION_POLICY`)、新 settings キー `worktree.bgIsolation`、`claude agents` に追加された `--allow-dangerously-skip-permissions` フラグを反映。ECC バージョン変更なし (v1.10.0 のまま)。
