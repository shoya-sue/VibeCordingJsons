## 変更詳細

### #1: 新環境変数 5 件 (README.md)

**対象行**: `### env (Environment Variables)` テーブル（現 L220–L236 付近）

追加変数:

| 変数名 | バージョン | 説明 |
|--------|-----------|------|
| `CLAUDE_CODE_SESSION_ID` | v2.1.132 | セッション ID（Bash サブプロセスの環境変数として自動設定） |
| `CLAUDE_CODE_DISABLE_ALTERNATE_SCREEN` | v2.1.132 | フルスクリーン alt-screen レンダラーを無効化し通常の端末スクロールバックを使用 |
| `CLAUDE_CODE_FORCE_SYNC_OUTPUT` | v2.1.129 | 同期出力を強制有効化（Emacs `eat` 等の自動検出が効かない端末向け） |
| `CLAUDE_CODE_PACKAGE_MANAGER_AUTO_UPDATE` | v2.1.129 | Homebrew/WinGet インストール時にバックグラウンドで自動アップグレード |
| `CLAUDE_CODE_ENABLE_GATEWAY_MODEL_DISCOVERY` | v2.1.129 | `/model` ピッカーでゲートウェイ `/v1/models` 探索を有効化（オプトイン） |

### #2: `skillOverrides` 設定キー (README.md)

**対象行**: `### Other Settings` テーブル（現 L240–L256 付近）

追加行:
```
| `skillOverrides` | スキルの表示制御（`off`: 完全非表示 / `user-invocable-only`: モデルには非表示 / `name-only`: 説明を折り畳み）（v2.1.129+） |
```

### #3: performance.md の env var 追記

**対象**: `CLAUDE_CODE_NO_FLICKER=1` の記載箇所に隣接して追加

追加行:
```
- **Disable alt-screen**: `CLAUDE_CODE_DISABLE_ALTERNATE_SCREEN=1` — alt-screen を無効化して通常スクロールバックを維持（v2.1.132+）
- **Force sync output**: `CLAUDE_CODE_FORCE_SYNC_OUTPUT=1` — Emacs eat 等で同期出力を強制有効化（v2.1.129+）
```
