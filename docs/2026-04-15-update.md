## 変更詳細

### #1: `/recap` コマンド追加（🟡 推奨）

Claude Code v2.1.108 で追加。長いセッションから戻ったときに「離席サマリー」を
手動で呼び出す新コマンド。`/config` でも有効化できる recap feature の手動トリガー。

**After (Slash Commands セクションに追加):**
```
- `/recap` — 離席後のセッションサマリーを手動表示
```

### #2: `ENABLE_PROMPT_CACHING_1H=1`（🟡 推奨）

Claude Code v2.1.108 で追加。プロンプトキャッシュの TTL を 5 分 → 1 時間に延長。
同一セッション内でシステムプロンプトのキャッシュが長持ちし、コスト削減になる。
API Key / Bedrock / Vertex / Foundry 全対応。`ENABLE_PROMPT_CACHING_1H_BEDROCK` の上位互換。

**Before:**
```json
"CLAUDE_CODE_NO_FLICKER": "1"
```

**After:**
```json
"CLAUDE_CODE_NO_FLICKER": "1",
"ENABLE_PROMPT_CACHING_1H": "1"
```
