## 変更詳細

### #1 `respondToBashCommands`（README + template/CLAUDE.md）

2.1.186 で `!` bash コマンドの出力に対し Claude が自動応答するようになった（旧来は context-only）。`"respondToBashCommands": false` で従来挙動に戻せる。

- README Other Settings テーブルに行を追加
- template/CLAUDE.md Important Notes に 1 行（既定 `true` / `false` で opt-out）

### #2 `teammateMode: "iterm2"`（README）

`teammateMode` に `iterm2` を追加。auto モードで `it2` CLI が見つからない場合は警告。README の display mode 列挙を `auto / in-process / tmux` → `auto / in-process / tmux / iterm2` に更新し `it2` 必須を注記。

### #3 SKILL.md frontmatter casing（root CLAUDE.md Conventions）

`display-name` / `default-enabled` / `fallback` / `metadata.*` キーが kebab-case / snake_case / camelCase いずれも受理。不正な YAML frontmatter は silent fail せず空メタデータで body をロード。root CLAUDE.md の SKILL.md 規約 bullet 群に追記。

### #4 `claude mcp login/logout`（template/CLAUDE.md Important Notes）

`/mcp` メニューを開かず CLI から MCP サーバー認証可能（`claude mcp login <name>` / `claude mcp logout <name>`）。`--no-browser` で stdin リダイレクトし SSH 越しに完了可。MCP 多用テンプレートに有用。

### #5 `CLAUDE_CODE_RETRY_WATCHDOG`（README env table）

`CLAUDE_CODE_MAX_RETRIES` は 15 が上限にキャップされた。unattended（CI/headless）セッションでは `CLAUDE_CODE_RETRY_WATCHDOG` を使う。README env テーブルに行を追加。

### #6 `/review <pr>`（template/CLAUDE.md Slash Commands）

`/review <pr>` が `/code-review medium` と同一レビューエンジンを使うようになった。slash command リストに追記し `/code-review` 系との関係を明示。

### #7 `Agent(<type>)` enforcement（README permission patterns）

`Agent(<type>)` deny ルールと `Agent(x,y)` allowed-types 制限が named subagent spawn でも実効化されるバグ修正。README の `Tool(param:value)` blockquote（`Agent(model:opus)` 例）に短く追記。
