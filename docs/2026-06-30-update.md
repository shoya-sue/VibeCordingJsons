## 破壊的変更チェック

Lens 5 トリガー語（hook matcher 意味論 / permission-rule matching / `mcp__*` tool-name matching / glob・regex / exact・substring / "now X-matches instead of Y-matches"）を 2.1.196 全項目に照合した結果、**該当項目なし**。

- セキュリティ項目「`claude mcp list`/`get` no longer spawn self-approved `.mcp.json` servers」は **trust/consent gating の変更**であって、hook matcher / permission-rule / glob-vs-regex の**マッチ意味論は変更していない**。テンプレ出荷の `template/.claude/settings.json` の hook matcher（v0.74.0 で `mcp__obsidian__.*` 正規形に統一済み）および `.mcp.json` の解釈は不変。
- 結論: **出荷 config に破壊的変更なし**。settings.json の matcher 編集も無いため `scripts/check-counts.sh` の bare-glob ガードに掛かる変更は発生しない（最終確認として実行する）。

## 変更詳細

### #1 `CLAUDE_ENABLE_STREAM_WATCHDOG`

> 公式 release notes（v2.1.196）: "The streaming idle watchdog is now on by default for all providers — it aborts and retries when a response stream produces no events for 5 minutes. Set `CLAUDE_ENABLE_STREAM_WATCHDOG=0` to disable."

- **README.md**（env var 表、`CLAUDE_CODE_DISABLE_BG_SHELL_PRESSURE_REAP` 行の後）に新規行:
  - `CLAUDE_ENABLE_STREAM_WATCHDOG` | streaming idle watchdog（応答ストリームが 5 分間イベント無しで abort+retry）。v2.1.196+ で全プロバイダ既定 ON（従来は一部のみ）。`0` で無効化 | `0`（無効化する場合）
- **template/CLAUDE.md**（Important Notes、MCP idle timeout 行の近く）に 1 行追加。

### #2 MCP self-approve セキュリティ

> 公式: "Security: `claude mcp list`/`get` no longer spawn `.mcp.json` servers that a repo self-approved via a committed `.claude/settings.json`; untrusted workspaces show `⏸ Pending approval`"

- **template/CLAUDE.md** 既存行（v2.1.186 `claude mcp login/logout`）に追補:
  - v2.1.196+ で `claude mcp list`/`get` は committed `.claude/settings.json` 経由で自己承認された `.mcp.json` サーバーを spawn しなくなった。untrusted workspace では `⏸ Pending approval` 表示。

### #3 Voice dictation 修正

> 公式: "Fixed voice dictation swallowing spaces and spuriously starting a recording during very fast typing when voice mode is enabled"

- **template/CLAUDE.md** voice 行（v2.1.195 のスペース無し言語 auto-submit 修正の直後）に追補。

### #4 Organization default models

> 公式: "Added support for organization default models — admins set it in the org console; it shows as \"Org default\" (or \"Role default\") in `/model` when you haven't picked one yourself"

- **template/CLAUDE.md** `/model` 行に 1 句追加。
- **README.md** `/model` セクション（v2.1.153+ 挙動変更の note 付近）に 1 行追加。
