### #1: README hook types カウント修正（🟡）

**原因**: CC v2.1.118 で `mcp_tool` ハンドラを追加した際、`hooks.md` と `CLAUDE.md` は更新したが
`README.md` の hooks セクション説明文を更新し忘れた。

**Before:**
```
4 hook types: `command` (shell), `http` (HTTP request), `prompt` (LLM judgment), `agent` (subagent)
```

**After:**
```
5 hook types: `command` (shell), `http` (HTTP request), `prompt` (LLM judgment), `agent` (subagent), `mcp_tool` (MCP tool invocation)
```

### #2: README MAX_MCP_OUTPUT_TOKENS 推奨値修正（🟡）

`settings.json` を `10000` に下げたが、README の env 変数テーブルは古い `25000`-`50000` のまま。

**Before:**
```
| `MAX_MCP_OUTPUT_TOKENS` | MCP output token limit | `25000`-`50000` |
```

**After:**
```

```
