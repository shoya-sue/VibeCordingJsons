## 変更詳細

### #1: `ANTHROPIC_BEDROCK_SERVICE_TIER` 環境変数（🟡）

**変更元 CC**: v2.1.122
**原因**: Amazon Bedrock 利用時にサービスティア（`default`/`flex`/`priority`）を指定できる
`ANTHROPIC_BEDROCK_SERVICE_TIER` 環境変数が追加された。
`X-Amzn-Bedrock-Service-Tier` ヘッダとして送信される。README の env テーブルに追記する。

**Before**: README の env テーブルに `ANTHROPIC_BEDROCK_SERVICE_TIER` の記述なし

**After**: README env テーブルに以下を追加:
```
| `ANTHROPIC_BEDROCK_SERVICE_TIER` | Bedrock サービスティア（`default` / `flex` / `priority`）（v2.1.122+） | `default` |
```
