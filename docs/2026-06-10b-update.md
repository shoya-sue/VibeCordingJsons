# VibeCording Update — 2026-06-10b

## 変更点サマリー

日本語の**音声入力（voice）環境**をテンプレート（＝グローバルインストール設定）に整備する。
Claude Code のネイティブ `/voice` ディクテーション（公式機能、v2.1.69+ / tap モード v2.1.116+）を
`settings.json` で**有効化＋日本語化**し、乱れた音声テキストを整形する `voice-input` スキルと、
完全オフラインのローカル whisper.cpp スクリプトを同梱する。手入力の負担軽減が目的。

## 環境スナップショット

- Local CC: `2.1.170`（`/voice` 完全対応）
- Local stack（shoya-sue 環境）: `whisper-cli`(whisper.cpp) / `ffmpeg` / `mlx` / `superwhisper` 導入済（Apple Silicon, macOS 26.5.1）
- Current release: `v0.56.0`（2026-06-10、ECC 2.0.0 移行）
- 一次情報: [Claude Code Docs — Voice dictation](https://code.claude.com/docs/en/voice-dictation)（WebFetch で実在・内容を検証済）

## 調査・検証

- リサーチ 2 体（macOS 日本語 STT ツール比較／Claude Code 音声操作ワークフロー）＋ claude-code-guide による公式 docs 照合の **3 独立ソースで一致**。
- ネイティブ `/voice` の存在は当初ハルシネーションを疑い、`claude --help`（voice 記述なし＝スラッシュコマンド）と **公式 docs を WebFetch で直接確認**して確定。
- 仕様（一次情報）: `/voice [hold|tap|off]`、Space タップで録音、無料（トークン非消費）、**クラウド処理**（音声を Anthropic へ送信）、**要 Claude.ai ログイン**（API キー/Bedrock/Vertex/Foundry/HIPAA は不可）、**SSH/Web 不可**、言語は `language`（`japanese`/`ja`）設定、`voice:pushToTalk` 再割当可。
