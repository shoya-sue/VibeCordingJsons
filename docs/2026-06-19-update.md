## 変更詳細

### #1 auto mode 破壊的 git/IaC blocking（CLAUDE.md）

CC 2.1.183 で auto mode が以下をビルトインでブロック:
- 破壊的 git: `git reset --hard` / `git checkout -- .` / `git clean -fd` / `git stash drop`（ユーザーが破棄を依頼していない場合）
- `git commit --amend`（当該セッションでエージェントが作成したコミット以外）
- `terraform destroy` / `pulumi destroy` / `cdk destroy`（対象スタックを明示しない場合）

既存の `template/CLAUDE.md:77` の auto mode 安全策行（v2.1.152+ / v2.1.158+ / v2.1.178+）の末尾に 1 文追記。テンプレートの permissions（settings.json deny/ask、AGENTS.md Denied Operations）とは別レイヤーの CC ビルトイン安全網である旨を含める。

### #3 `attribution.sessionUrl`（README）

web / Remote Control セッションで commits / PRs から claude.ai セッションリンクを省略する新サブキー。テンプレートの `settings.json` には既に `attribution.commit` / `attribution.pr` があるが、`sessionUrl` は web/RC 限定機能のため **settings.json には追加せず**、README settings table の `attribution` 行に言及のみ追記。

### #4+5 `/config --help` + ピッカー toggle 挙動（CLAUDE.md）

- `/config --help`: `/config key=value` で指定可能な shorthand キー一覧を表示。
- ピッカー挙動変更: Enter / Space の両方で選択中設定を変更、Esc は revert ではなく保存して閉じる。

`template/CLAUDE.md:67` の `/config [key=value]` 行（v2.1.181+ で `/config key=value` を doc 済み）に統合追記。
