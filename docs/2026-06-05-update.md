## 変更詳細

### #1 hooks.md — Stop/SubagentStop `additionalContext` (v2.1.163+)

Advanced Features 末尾(`background_tasks` / `session_crons` セクションの後)に新サブセクションを追加:

Stop / SubagentStop フックが `hookSpecificOutput.additionalContext` を返すと、
エラーラベルなしで Claude にフィードバック(追加コンテキスト)を渡せる。
従来の block(`decision: "block"` + reason)はエラー扱いの表示になるが、こちらは非エラーで文脈注入のみ。

### #2 README.md — `requiredMinimumVersion` / `requiredMaximumVersion`

Other Settings 表に 1 行追加(managed settings、バージョン互換性の強制)。

### #3 README.md — `CLAUDE_CODE_SESSION_ID` 行追記

before: 「v2.1.154+ で MCP stdio サーバーにも `CLAUDE_CODE_SESSION_ID` と `CLAUDECODE=1` が渡る」
after: 末尾に「`--resume` 時も stdio MCP サーバーに渡る(v2.1.163+)」を追記。

### #4 `/effort` 永続化 (v2.1.162+)

- `template/CLAUDE.md` L46: `/effort` 行に「選択したレベルは v2.1.162+ で新セッションにも引き継がれる」を追記
