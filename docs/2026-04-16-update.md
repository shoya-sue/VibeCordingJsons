## 変更詳細

### #1: extended-thinking 改善インジケーター（⚪ スキップ）

**対象バージョン**: CC v2.1.109
**カテゴリ**: UI 変更

長い操作中に rotating progress hint が表示されるようになった。設定変更不要。

---

### #2: `autoScrollEnabled`（⚪ スキップ）

**対象バージョン**: CC v2.1.110
**カテゴリ**: settings.json

フルスクリーンモードでの会話自動スクロールを無効化するオプション。

**After (version gate が解除されたら追加):**
```json
"autoScrollEnabled": false
```

---

### #3: `/tui` コマンド（⚪ スキップ）

**対象バージョン**: CC v2.1.110
**カテゴリ**: Slash Command

`/tui fullscreen` でチラつきなし alt-screen レンダリングを同一会話内で起動可能。
現在テンプレートでは `CLAUDE_CODE_NO_FLICKER=1` env var で同等機能を提供中。

---

### #4: `/focus` コマンド（⚪ スキップ）

**対象バージョン**: CC v2.1.110
**カテゴリ**: Slash Command / Behavior Change

v2.1.110 で `Ctrl+O` の役割が分割された:
- `Ctrl+O` → verbose transcript のみ
- `/focus` → フォーカスビューの toggle（新コマンド）

`performance.md` の `Ctrl+O` 記述は v2.1.110 以降でも引き続き正確（verbose = thinking output）。
`/focus` コマンドは version gate が解除されたら Slash Commands に追加する。

---

### #5: `TRACEPARENT`/`TRACESTATE`（⚪ スキップ）

**対象バージョン**: CC v2.1.110
**カテゴリ**: 環境変数

SDK/headless モードでの分散トレーシング用 env var。通常テンプレートユーザーには不要。

---

### #6: `CLAUDE_CODE_ENABLE_AWAY_SUMMARY=0`（⚪ スキップ）

**対象バージョン**: CC v2.1.110
**カテゴリ**: 環境変数

session recap をテレメトリ無効ユーザーでデフォルト有効化、opt-out は `=0` で可能。
CLAUDE.md に `/recap` が既にドキュメント済みのため追加不要。

---

### #7-#8: バグ修正・ツール動作変更（⚪ スキップ）

**対象バージョン**: CC v2.1.110
**カテゴリ**: バグ修正

- `PermissionRequest` フックの `updatedInput` が `permissions.deny` で正しく再チェックされるように修正
- `PreToolUse` の `additionalContext` がツール呼び出し失敗時にドロップされる問題を修正
- Write ツールが IDE diff で手動編集されたことをモデルに通知するように変更
- Bash ツールのタイムアウト強制（ドキュメント上の最大値を超えた値を拒否）
- MCP ツール呼び出しの SSE/HTTP 切断時の無限ハング修正
- 非ストリーミングフォールバックリトライによる数分ハング修正

設定変更なし。テンプレートへの反映不要。

---

**対象バージョン**: ECC v1.10.0
**カテゴリ**: ECC
