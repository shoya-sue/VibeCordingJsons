## 変更詳細

### #9 — hooks matcher 構文の gotcha 注記（`hooks.md`）

Claude Code 2.1.191 で「カンマ区切り matcher（例 `"Bash,PowerShell"`）がサイレントに一度も発火しない」バグが修正された。テンプレート自身の `settings.json` hooks は既に pipe 区切り（`"Write|Edit"`）を使っているため修正不要だが、テンプレ利用者が自前 hook を書く際の footgun になりうるため、events テーブル直後に matcher 構文の注記を追加する。

**Before:** events テーブルの直後に `## Advanced Features` が続く（matcher 構文の明示注記なし）。

**After:** events テーブルと `## Advanced Features` の間に以下の blockquote を挿入:

> **マッチャー構文**: 複数ツールを 1 つの hook にまとめるときは **パイプ区切り**（`"Write|Edit"`）を使う。カンマ区切り（`"Bash,PowerShell"`）は v2.1.191 未満で **サイレントに一度も発火しない** バグがあった（v2.1.191 で修正）。古い CC でも確実に動かすならパイプ区切りに統一する。
