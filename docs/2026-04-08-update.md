## 変更詳細

### #1: `/powerup` コマンド追加（v2.1.90）

Claude Code の対話型学習コマンド。インタラクティブなレッスン・チュートリアルを起動する。

**full/CLAUDE.md の Slash Commands セクションに追加:**
```
- `/powerup` — インタラクティブな学習レッスンを起動
```

**full/.github/copilot-instructions.md のスラッシュコマンド表にも追加。**

---

### #2: PreToolUse `"defer"` 権限決定（v2.1.89）

PreToolUse フックがツール実行を「defer（保留）」として返せるようになった。通常の allow/block に加え、ユーザー判断を仰ぐ中間状態。

**hooks.md の Advanced Features セクションに追記:**

```jsonc
// PreToolUse で "defer" を返すと権限確認を通常フローに委ねる
{
  "decision": "defer"
}
```

---

### #3: `hookSpecificOutput.sessionTitle`（v2.1.94）

UserPromptSubmit フックの出力に `sessionTitle` フィールドを返すことでセッションタイトルを設定できる。

**hooks.md の Advanced Features セクションに追記:**

```jsonc
// UserPromptSubmit フックで sessionTitle を設定
{
  "hookSpecificOutput": {
    "sessionTitle": "My Session Title"
  }
}
```

---

### #4: `disableSkillShellExecution` 設定（v2.1.91）

スキル / スラッシュコマンド / プラグインコマンド内のインライン Shell 実行を無効化するセキュリティ設定。

**full/.claude/settings.json の設定コメントとして追加（デフォルト false なので明示的に記載のみ）:**

```json
"disableSkillShellExecution": false
```

---

### #5: `sandbox.failIfUnavailable` 設定（v2.1.89）

sandbox が有効だが利用不可な環境では即座に終了するフェイルセーフ設定。

**full/.claude/settings.json の sandbox セクションに追加:**

```json
"sandbox": {
  "failIfUnavailable": false
}
```
