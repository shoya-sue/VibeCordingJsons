# VibeCording Update — 2026-05-15

## 変更点サマリー

Claude Code v2.1.142 (2026-05-14) の変更を反映。Fast mode のデフォルトが Opus 4.7 に切替（Opus 4.6 にピン留めする `CLAUDE_CODE_OPUS_4_6_FAST_MODE_OVERRIDE` 環境変数を追加）、リモート HTTP/SSE MCP サーバーで `MCP_TOOL_TIMEOUT` が機能するようになった旨を README env テーブルに追加、`claude agents` の新フラグ（`--add-dir`/`--settings`/`--mcp-config`/`--plugin-dir`/`--permission-mode`/`--model`/`--effort`/`--dangerously-skip-permissions`）を AGENTS.md に追記、`SessionStart`/`Setup`/`SubagentStart` フックは `command` 型のみという制約を hooks.md に明記。ECC バージョン変更なし。
