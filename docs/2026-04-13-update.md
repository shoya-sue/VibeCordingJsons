## 変更詳細

### #1: CLAUDE_CODE_SUBPROCESS_ENV_SCRUB=1（🔴 必須）

Claude Code v2.1.101+ でサブプロセス実行時に認証情報（API keys, tokens）を
環境変数から自動削除するセキュリティ強化。

**Before:**
```json
"env": {
  "ECC_HOOK_PROFILE": "standard",
  "ECC_DISABLED_HOOKS": ""
}
```

**After:**
```json
"env": {
  "ECC_HOOK_PROFILE": "standard",
  "ECC_DISABLED_HOOKS": "",
  "CLAUDE_CODE_SUBPROCESS_ENV_SCRUB": "1"
}
```

### #2: CLAUDE_CODE_NO_FLICKER=1（🟡 推奨）

v2.1.91+ で追加されたちらつきなし alt-screen レンダリング。
視覚的な快適さの向上。

**After（#1に追加）:**
```json
"CLAUDE_CODE_NO_FLICKER": "1"
```
