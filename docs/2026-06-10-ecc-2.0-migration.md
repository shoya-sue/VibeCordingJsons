# VibeCording Update — 2026-06-10 (ECC 2.0.0 migration)

## 変更点サマリー

ECC を **1.10.0 → 2.0.0** に移行。ECC 2.0.0 は単なる version bump ではなく **plugin の RENAME**（`everything-claude-code` → `ecc`）を伴うため、テンプレート全体の ECC 連携を書き換えた。marketplace / GitHub repo 名は `everything-claude-code` のまま、**plugin 名と agent namespace のみ `ecc`** に変わる。

> ⚠️ 導入前に確認: ECC 2.0.0 は major リリース。本移行はローカルに `ecc@everything-claude-code` 2.0.0 を実インストールし、全互換性チェックが pass したことを確認した上で実施。旧 `everything-claude-code` 1.10.0 plugin は disabled のまま残る（手動 uninstall は任意）。

## 環境スナップショット

- Local ECC installed: `ecc 2.0.0`（`~/.claude/plugins/cache/everything-claude-code/ecc/2.0.0`、**enabled**）+ legacy `everything-claude-code 1.10.0`（disabled）
- Template `ECC_PLUGIN_ROOT`: `.../everything-claude-code/ecc/2.0.0`（移行後）
- Previous release: `v0.55.0`（同日、Fable 5 docs）

## 変更詳細

### 1. install.sh（ECC auto-detect）
- `ECC_CACHE` を `.../everything-claude-code/everything-claude-code` → `.../everything-claude-code/ecc` に変更
- sed regex を `/everything-claude-code/ecc/<version>` にマッチするよう更新

### 2. template/.claude/settings.json
- `ECC_PLUGIN_ROOT`: `.../everything-claude-code/everything-claude-code/1.10.0` → `.../everything-claude-code/ecc/2.0.0`
- hooks（5 個の `${ECC_PLUGIN_ROOT}/scripts/hooks/...`）は path 構造が同一のため**変更不要**

### 3. agent namespace `everything-claude-code:<agent>` → `ecc:<agent>`（31 箇所）
- `README.md`（4）/ `template/README.md`（4）/ `template/.claude/rules/subagent-delegation.md`（delegation matrix 11 + prose 2）/ `template/.claude/agents/*.md`（6 override の description）

### 4. install コマンド `everything-claude-code@everything-claude-code` → `ecc@everything-claude-code`
- `README.md` / `template/README.md` / `template/AGENTS.md`（marketplace add `affaan-m/everything-claude-code` は不変）

### 5. plugin-provided counts 38/156/72（1.10.0）→ 64/261/84（2.0.0）+ 版表記 ECC 1.10.0 → 2.0.0
- `README.md` / `template/README.md`（What's Included 表 + ECC Integration）/ `template/AGENTS.md`

### 6. その他
- `template/CLAUDE.md` / `template/AGENTS.md` の「Requires `everything-claude-code` plugin」→「Requires the `ecc` plugin（everything-claude-code marketplace）」
- `update-release` SKILL の preflight を `ecc/` + legacy 両走査に更新、例示版を 2.0.0 に
- `scripts/check-counts.sh` の STALE guard に旧 counts（`38 agents` / `156 skills` / `72 commands`）+ 旧 install cmd を追加（再発防止）
